# TDSimulator - 交易模拟器

一个基于 Flask 的交易模拟器，支持基础模型和高级模型两种交易模拟方式。

## 功能特性

### 基础模型

- **价格调节器**：垂直滑块，支持 400-450 价格范围调节
- **权益面板**：实时显示当前价格、盈亏百分比、权益比率
- **高级选项面板**：鼠标悬停右上角按钮展开
  - 买多/卖空切换
  - 保证金率滑块（1%-100%）
  - 杠杆倍数显示（自动计算）
- **买入价格输入**：可调整开仓价格
- **重置按钮**：恢复初始状态

### 高级模型

- **价格图表**：Canvas 绘制，支持鼠标划线绘制价格走势
- **权益图表**：自动联动绘制对应权益轨迹
- **交易方向**：买多/卖空切换
- **保证金率**：滑块调节，自动计算杠杆
- **统计面板**：
  - 当前价格
  - 当前权益
  - 最高/最低权益
  - 总收益率
- **清除画线**：一键清除重新开始

## 计算公式

### 权益计算

```
杠杆 = 1 / 保证金率
价格变化率 = (当前价格 - 开仓价格) / 开仓价格

买多权益 = 1 + 杠杆 × 价格变化率
卖空权益 = 1 - 杠杆 × 价格变化率
```

### 示例

| 保证金率 | 杠杆倍数 | 价格变化 | 买多权益 | 卖空权益 |
|---------|---------|---------|---------|---------|
| 10% | 10倍 | +5% | 1.5 (+50%) | 0.5 (-50%) |
| 10% | 10倍 | -5% | 0.5 (-50%) | 1.5 (+50%) |
| 20% | 5倍 | +5% | 1.25 (+25%) | 0.75 (-25%) |

## 环境要求

- Python 3.7+

## 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/yackbird/TDSimulator.git
cd TDSimulator
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行应用

```bash
python app.py
```

### 4. 访问应用

打开浏览器访问：http://localhost:5000
