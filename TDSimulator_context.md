# TDSimulator 项目上下文

## 项目目标
创建一个**交易模拟器**网站，包含价格调节棒和权益面板的实时联动。

---

## 核心需求

### 功能
1. **价格调节棒**：垂直滑块，用户可拖动调节价格（400-450范围）
2. **权益面板**：实时显示计算结果
   - 当前价格
   - 盈亏百分比
   - 权益比率
3. **买入价格输入**：可调整买入价格
4. **重置按钮**：恢复初始状态

### 计算公式
```
盈亏% = (当前价格 - 买入价格) / 买入价格 × 100
权益比率 = 当前价格 / 买入价格
```

## 样式要点

- 背景：深蓝渐变 `linear-gradient(135deg, #1a1a2e, #16213e, #0f3460)`
- 卡片：毛玻璃效果 `backdrop-filter: blur(10px)`
- 滑块轨道：红黄绿渐变（绿=高价盈利，红=低价亏损）
- 盈利显示：绿色 `#27ae60`
- 亏损显示：红色 `#e74c3c`

---

## 交互逻辑

1. 用户拖动滑块 → 实时更新价格
2. 价格变化 → 实时计算盈亏和权益比率
3. 修改买入价格 → 重新计算所有指标
4. 点击重置 → 恢复初始状态（价格420，买入价420）
